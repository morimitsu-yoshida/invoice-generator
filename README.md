# 請求書発行ツール

MY造園（吉田 盛隆）用の請求書HTML発行ツール。サンプル `IMG_1057.jpg` のレイアウトを再現。

## 使い方

1. `index.html` をダブルクリックでブラウザで開く
2. 左フォームに入力 → 右にリアルタイムプレビュー
3. 「印刷 / PDF保存 (Cmd+P)」で印刷ダイアログ → 「PDFとして保存」も可
4. 「下書き保存」で内容をブラウザに保存、「下書き呼出」で復元

## 機能
- 請求日デフォルト = 今日
- 自社情報・振込先は事前入力済み（必要なら編集）
- 明細行は追加/削除可、金額は数量×単価で自動計算
- 小計・消費税・合計は自動。差額調整も入力可
- 印刷時はフォーム非表示、A4プレビューのみ印刷
- 下書きはブラウザ localStorage に保存
